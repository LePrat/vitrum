## Install
To get started, download the project files using one of the methods below:

#### Option A: Download ZIP (Easiest)
If you just need the current version of the files, click the green Code button at the top of this page and select Download ZIP. Extract the folder once the download finishes.

#### Option B: Clone via Git
If you have Git installed and want to pull updates easily, run the following command in your terminal:

```Bash
git clone https://github.com/LePrat/vitrum.git
```
